# Form

Form is the top-level container that powers forms. It provides a context for all form elements to interact with.

```jsx
import { Form } from '@ryuuji3/react-forms';
```

## Usage

```jsx
<Form onSubmit={handleSubmit}>
    {/* Form elements go here */}

    <button type="submit">Submit</button>
</Form>
